## [1.0.0] - 2026-02-04

### 新增

- 🎉 初始版本发布
- ✨ AiTableModules 主表格组件
  - 表格展示功能
  - 列表搜索功能
  - 分页功能
  - 排序功能
  - 表头自定义配置
  - 批量操作功能
- ✨ AddReuse 抽屉表单组件
  - 新增数据功能
  - 编辑数据功能
  - 表单验证功能
  - 自定义表单配置
- ✨ SearchForm 搜索表单组件
- ✨ ImportComponent 数据导入组件
- ✨ TagInput 标签输入组件
- ✨ SettingsHeaders 表头设置组件
- ✨ ActionButtons 操作按钮组件
- ✨ MultiTooltips 多工具提示组件
- ✨ SimpleDetail 简单详情组件
- 📦 支持按需引入
- 📦 支持完整引入
- 🎨 基于 Element Plus 的现代化 UI
- 📝 完整的 README 文档

### 依赖

- Vue 3.3.0+
- Element Plus 2.0.0+
- Sortablejs 1.15.6
- Dayjs 1.11.19
